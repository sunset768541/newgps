欢迎使用定位软件
--------------------------------------------------------------------------------------------
该软件的功能包括
--------------------------------------------------------------------------------------------
定位功能：提供GPS、基站、Wi-Fi等多种定位方式，适用于室内、外多种定位场景；
计算工具：包括距离计算、坐标转换、调起百度地图导航等功能；
基础地图：包括基础地图、卫星图、路况图和各种覆盖物，及与地图相关的操作、事件监听；个性化地图
好友功能:可以添加好友并动态监听好友的位置变化
越界提示：可以在地图上画出界限，当被监听者移动位置越界的时候发出提示
足迹记录: 可以存储自己的足迹
发表评论：可以对一个地方发表评论
