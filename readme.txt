欢迎使用定位软件
--------------------------------------------------------------------------------------------
该软件的功能包括
--------------------------------------------------------------------------------------------
定位功能：提供GPS、基站、Wi-Fi等多种定位方式，可以显示卫星地图和2维地图；
计算工具：包括距离计算、面积、坐标转换功能；
基础地图：包括基础地图、卫星图、及与地图相关的操作、事件监听；
好友功能: 可以添加好友并动态监听好友的位置变化
越界提示：可以在地图上画出界限，监听自己位置的变化
足迹记录: 可以存储自己的定位信息到服务器的数据库
